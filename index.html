💾 Sistema de Backup — 3 Camadas
☁️ Camada 1 — Google Drive (Nuvem)
Backup automático a cada 3 minutos. Configure em Configurações → Google Drive.
Restaure em qualquer computador. Arquivo: PetShopColosso_Backup.json
💜 Camada 2 — Pendrive / Pasta Local
Clique em 'Salvar no Pendrive' na barra lateral. Salva automaticamente a cada 5 minutos.
Suporta múltiplos pendrives. Gera .json + .xlsx com data.
💾 Camada 3 — Navegador (localStorage)
Salvo automaticamente a cada alteração.
⚠️ ATENÇÃO: Limpar cache do navegador apaga estes dados! Mantenha Drive ou Pendrive ativo!
⚠️ SITUAÇÕES DE RISCO
Trocou de PC? → Restaure do Google Drive ou carregue do Pendrive
Formatou o PC? → Restaure do Google Drive (dados na nuvem são preservados)
Perdeu o pendrive? → Restaure do Google Drive
Sem internet? → localStorage + pendrive funcionam offline normalmente

function buildMsg(template, vars) {
  let msg = template;
  Object.entries(vars).forEach(([k,v]) => msg = msg.replaceAll(`{${k}}`, v));
  return msg;
}
